## Common Questions

### Installation Issues

**Problem**: SDK not loading or initializing

**Solutions**:
1. Check browser console for errors
2. Verify API key is correct
3. Ensure script is loaded before `AdTruth.init()`
4. Check CORS settings if self-hosting

See [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md#troubleshooting) for more.

### Integration Issues

**Problem**: Data not appearing in dashboard

**Solutions**:
1. Verify tracking script is on all pages
2. Check Network tab for successful API calls
3. Wait 5-10 minutes for data to process
4. Check API key matches your website

See platform-specific guides in [examples/](examples/).

### Performance Concerns

**Problem**: Script slowing down website

**Solutions**:
1. Verify you're using the minified version (12KB)
2. Check for duplicate script tags
3. Ensure async loading
4. Review browser console for errors

See [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md#performance-optimizations).

## Security Issues

**DO NOT** report security vulnerabilities through public issues.

Please follow our security policy:

**[Security Policy](SECURITY.md)**

Report vulnerabilities to:
- **GitHub Security Advisories**: [Report a vulnerability](https://github.com/papa-torb/adtruth/security/advisories/new)
- **Email**: See [SECURITY.md](SECURITY.md) for contact

Have an idea for a new feature?

1. Check [existing feature requests](https://github.com/papa-torb/adtruth/issues?q=is%3Aissue+is%3Aopen+label%3Aenhancement)

## Response Times

This is an open-source project maintained by volunteers:

- **Bug Reports**: We aim to respond within 1-3 business days
- **Feature Requests**: Response within 1 week
- **Security Issues**: Response within 48 hours (see [SECURITY.md](SECURITY.md))
- **Pull Requests**: Review within 1 week

*Note: Response times are best-effort and may vary.*
