The ground truth collection system is now fully integrated into the standard tracking flow. Every page view automatically gets analyzed for behavioral impossibilities.

How It Works

  1. SDK collects behavioral data (clicks, mouse movement, scroll, time on page)
  2. SDK sends data to /track endpoint (no special configuration)
  3. Backend automatically detects impossibilities from behavioral data
  4. Backend computes ground truth label (bot, human, suspicious, unknown)
  5. Backend stores everything in page_views table with confidence scores
